# Conversor de Monedas en C

Programa de consola que permite convertir montos entre quetzales (Q), dólares (USA) y euros (€), realizado por copilot.

## Prerrequisitos

Compilador GCC instalado en una distribución de Linux.

## Pasos para su ejecución:

1. Abrir la terminal en la carpeta del proyecto
2. Compilar el código, con el siguiente comando: gcc conversor.c -o conversor
3. Ejecutar el programa, con el siguiente comando: ./conversor
